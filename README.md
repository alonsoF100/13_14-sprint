# TodoApp - бэкенд для приложения задач

## Описание проекта
Бэкенд-сервер для управления списком задач с поддержкой повторяющихся событий. Реализован на Go с использованием SQLite для хранения данных.

## Выполненные задания со звёздочкой
- Поддержка порта через переменную окружения TODO_PORT
- Поддержка пути к БД через переменную окружения TODO_DBFILE

## Сервер
